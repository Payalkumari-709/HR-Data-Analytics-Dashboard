# 📊 HR Analytics Dashboard | PostgreSQL + Power BI

An end-to-end **HR Analytics** project that analyzes employee workforce data using **PostgreSQL** for data cleaning and SQL analysis, and **Power BI** for interactive dashboard visualization.

The project focuses on employee demographics, workforce distribution, hiring trends, departmental analysis, and employee tenure to generate meaningful business insights for HR decision-making.

---

## 🚀 Project Overview

This project demonstrates the complete data analytics workflow:

- Data Import
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Business Insights
- Interactive Dashboard Development

The objective is to help HR teams understand workforce composition, hiring patterns, employee distribution, and departmental performance.

---

## 🛠️ Tools & Technologies

- PostgreSQL
- SQL

## 📂 Dataset

**Human Resources Employee Dataset**

The dataset contains employee information including:

- Employee ID
- Gender
- Birthdate
- Race
- Department
- Job Title
- Location
- Hire Date
- Termination Date
- City
- State

---

# 📋 Data Cleaning & Preprocessing

The following preprocessing steps were performed using PostgreSQL:

- Renamed the Employee ID column
- Converted Birthdate from TEXT to DATE
- Converted Hire Date from TEXT to DATE
- Converted Termination Date from TEXT to DATE
- Created an Age column
- Removed invalid date values
- Standardized date formats
- Verified data types

---

# 📈 SQL Analysis

The following business questions were answered using SQL:

1. Gender breakdown of employees
2. Race distribution of employees
3. Age group distribution
4. Headquarters vs Remote employees
5. Average employee tenure
6. Gender distribution across departments
7. Job title distribution
8. Department-wise termination rate
9. Employee distribution by state
10. Hiring trend over the years
11. Average tenure by department
